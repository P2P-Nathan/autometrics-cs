--------------------------------------------------------
---------- Autometrics.Instrumentation Readme ----------
--------------------------------------------------------

Autometrics.Instrumentation is a C# library for the Autometrics observability micro-framework. It simplifies the instrumentation of your code to collect key metrics including function call counts, durations, and build information. 

To get started:

1. Install this package to your project via NuGet package manager.
2. Use the [Autometrics] attribute to instrument your methods.
3. Setup and exporter for the metrics.  Otlp, Console, etc.

For detailed instructions and examples, visit our GitHub repository: https://github.com/autometrics-dev/autometrics-cs
